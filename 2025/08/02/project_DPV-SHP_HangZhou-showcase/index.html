杭州二手房数据处理及可视化 项目展示

上次更新: 2025-08-02 21:13:34

项目简介

这是一个基于 Vue3 + ECharts 开发的「杭州二手房数据处理及可视化」项目，展示了杭州二手房市场的价格、区域、户型等多维度数据的分析与可视化呈现。

项目特点

  • ✨ 多维度数据分析：涵盖价格、面积、房型、楼层、朝向、区域分布等
  • 🎨 交互式图表展示：支持筛选、缩放、联动、钻取等交互操作
  • 📊 数据处理与可视化一体化：从 Excel 原始数据筛洗 → JSON格式转换 → 根据图表需求处理数据 → 图表展示
  • 🚀 现代化技术栈：基于 Bun + Vue3 + TypeScript 构建，开发构建体验流畅

技术栈

  • 前端框架：Vue 3 + TypeScript
  • 图表库：ECharts 5
  • 构建工具：Vite
  • 数据处理：Python
  • 数据格式：JSON

项目功能

主要功能模块

  1. 各区域房源单价分布

    • 展示杭州不同区域的房源价格分布情况，使用箱线图展示各区域价格的中位数、四分位数及异常值
    • 支持横向比较不同区域的价格水平

  2. 关注度前十房源的关注人数

    • 展示关注度最高的前十套房源及其关注人数
    • 使用柱状图直观展示不同房源的关注度差异

  3. 不同装修类型房源单价

    • 展示毛坯、简装、精装等不同装修类型房源的价格分布
    • 使用散点图展示各装修类型房源的价格区间及分布密度

  4. 各区域房源数量占比

    • 展示杭州各区域房源数量的占比情况
    • 使用饼图直观展示不同区域的房源分布比例
    • 配套图例说明各颜色对应的区域

  5. 发布时间与关注人数变化

    • 展示房源发布时间与关注人数的变化趋势
    • 使用折线图展示不同发布时间段房源的关注人数变化
    • 帮助分析房源发布时间对关注度的影响
